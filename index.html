DLAB Practice — Zorathi Language
One new rule per set. Start at Level 1. Build up naturally.
💡 What is this and what do I do?
The DLAB is a military test that measures how fast you can learn a brand-new language. On the real test, they give you a made-up language that nobody on Earth speaks, teach you its rules in a few minutes, then immediately quiz you. They use a fictional language so that nobody has a head start — a Russian speaker and someone who only speaks English both start at zero.
These exercises work exactly the same way. We use a made-up language called Zorathi. Each set below teaches you one new rule, shows you examples, then gives you 3 practice questions. Start at Level 1, Set 1 — even if it looks easy. Every set builds on the one before it.
How to answer each question
1If the question shows a Zorathi word and asks what it means — type the English meaning.
2If the question gives you an English word and asks you to translate — type the Zorathi version using the rule shown above the question.
